## 4. 코딩과 HTML
 - 사람이 하는 일 (원인)
    - 코드 (code): 부호 또는 신호
    - 소스 (source): 원천
    - 언어: 약속
- 기계가 하는 일 (결과)
  - 애플리케이션 (=앱)
  - 응용 프로그램
  - 웹 페이지 < 웹 사이트 < 웹 애플리케이션
  
- HTML이 중요한 2가지 이유
  1. 쉽다.
  2. 매일 사용한다.
  
- 퍼블릭 도메인 (Public Domain)
  : 저작권이 없는 것
  - 예) 한글, 알파벳, 웹

## 6. 기본 문법 - 태그
 - `<strong> </strong>`: 진한 글씨
 - `<u> </u>` : 밑줄
 
 - 한글이 깨질 경우
   - `<meta charset="utf-8">`
   
## 7. 혁명적인 변화
 - `<h1> </h1>` ~ `<h6> </h6>` : 제목 (heading), 글자의 크기 및 두께 지정
